# Learning_Flutter

현재 터미널 창에 대해서만 PATH 변수를 설정 <br>
export PATH="$PATH:`pwd`/flutter/bin"
**pwd 는 작은따옴표**

현 상태 확인 <br>
flutter doctor

앱 생성  <br>
flutter create A <br>
cd A

앱 실행 <br>
flutter devices <br>
flutter run <br>
