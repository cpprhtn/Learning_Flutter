# Learning_Flutter

현재 터미널 창에 대해서만 PATH 변수를 설정 <br>
```
PATH="$PATH:`pwd`/flutter/bin"
```

현 상태 확인 <br>
flutter doctor

앱 생성  <br>
flutter create A <br>
cd A

ios simulator <br>
open -a Simulator <br>

앱 실행 <br>
flutter devices <br>
flutter run <br>
flutter run --no-sound-null-safety
